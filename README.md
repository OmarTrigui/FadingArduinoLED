# Arduino - SparkFun Electronics (@ENIS Project)

## Fading RGB LED Colors with LCD display ( RGB color codes )
### Required hardware :
```
- Arduino
- 330ohm resistors
- RGB LED
- BreadBoard
- Wires
- Liquid Crystal Display
- Rotary Potentiometer ( To adjust display's brightness )
```
